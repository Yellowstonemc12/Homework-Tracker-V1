YS Tracker V1 is a web-based platform designed to help teachers manage and monitor student homework efficiently. Instead of relying on spreadsheets or paper records, teachers can quickly assign homework, track outstanding tasks, and maintain a complete history of completed work in one simple interface.

The system allows teachers to import student lists using CSV files, making setup fast and convenient. Teachers can assign homework to students using either index numbers or names, with real-time student identification to reduce errors and improve accuracy.

Key features include:

|  📂 CSV student list import
|  
|  📝 Homework assignment and tracking
|  
|  🔍 Instant student identification and lookup
|  
|  ⚡ Priority levels for homework records
|  
|  ✅ One-click completion tracking
|  
|  📜 Automatic history archiving
|  
|  🗑️ History management tools
|  
|  👩‍🏫 Separate data for each teacher account

Homework Tracker is built with simplicity in mind, providing an intuitive experience that requires minimal training while helping teachers stay organized and save time.

Current Version: V1 (Beta)

Future Versions: V2 and V3 will introduce advanced analytics, reporting tools, classroom insights, and premium features for schools and educators.
